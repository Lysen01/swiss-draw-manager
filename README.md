# Swiss Draw Manager

Веб-застосунок для турнірів за спрощеною швейцарською системою.

## Можливості

- Створення турніру та налаштування кількості турів.
- Додавання учасників з рейтингами.
- Автоматичне формування пар на кожен тур.
- Уникання повторних пар (коли можливо).
- Баланс кольорів (білі/чорні) та уникнення довгих серій кольору.
- Автоматичний BYE при непарній кількості гравців.
- Внесення результатів і побудова таблиці.
- Додаткові коефіцієнти: Buchholz, Sonneborn-Berger.
- Автозбереження стану турніру у localStorage.

## Запуск

Відкрийте `index.html` у браузері.

## Обмеження

Це не повна реалізація офіційних FIDE/FMJD правил жеребкування, а практична версія для локальних турнірів.
